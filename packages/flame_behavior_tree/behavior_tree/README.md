Behavior tree is a very common way of implementing AI behavior in game and robotics. This package
provides a simple and easy to use behavior tree API in pure dart.


## Features

- Nodes
  - Composite
    - Sequence
    - Selector
  - Decorator
    - Inverter
    - Limiter
